# Social Media Mental Health Analysis

## 📌 Project Overview
This project analyzes the relationship between social media usage and mental health. 
The goal is to identify patterns and understand how factors like screen time, sleep, and usage behavior impact mental well-being.

---

## 🎯 Objectives
- Analyze social media usage patterns  
- Identify factors affecting mental health  
- Build a predictive model using regression  
- Extract meaningful insights from data  

---

## 📂 Dataset
- **Rows:** 481 
- **Columns:** 21  
- **Source:** Kaggle (Students Social Media Addiction dataset)  

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

## 🔍 Key Analysis
- Data cleaning and preprocessing  
- Handling missing values  
- Feature analysis  
- Correlation between variables  

---

## 📊 Visualizations
- Age distribution  
- Gender analysis  
- Correlation heatmap  
- Screen time vs mental health  

---

## 🤖 Machine Learning
- **Model Used:** Linear Regression  
- **R² Score:** 0.331494
- **Mean Squared Error:** 1.036
  

---

## 📌 Conclusion
- Increased social media usage is linked with higher stress levels  
- Sleep plays a significant role in mental health  
- The model can predict mental health trends effectively  
- Further improvements can be made using advanced models
